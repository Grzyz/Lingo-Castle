# Lingo-Castle
Tingkatkan skill bahasa Inggris dengan metode progresif! Lingo Castle menggabungkan desain estetik dengan fitur cerdas: Replay Marathon untuk latihan tanpa tekanan, dan Smart Review untuk memperbaiki kesalahan secara fokus. Pantau progresmu, raih XP, dan ubah rank dari Pengembara menjadi Sang Penakluk Kamus
